# Audi KPI Tree v3 — Looker Studio Community Visualization

Custom community visualization for Looker Studio that renders the Audi brand performance KPI Tree (5 pillars, 9 sub-categories, 29 KPIs with 3 visual variants).

## Use in Looker Studio

1. Open your Looker Studio report in edit mode
2. **Add a chart** → scroll down → **Community visualizations and components**
3. Click **+ Build your own visualization**
4. Paste this manifest path:
   ```
   gs://flowerelon.github.io/audi-kpi-viz/
   ```
   Or the full HTTPS URL:
   ```
   https://flowerelon.github.io/audi-kpi-viz/
   ```
5. Drag the **Audi KPI Tree v3** viz onto your canvas
6. Map the 9 data fields in the right panel

See [README-DEPLOY.md](../README-DEPLOY.md) for full setup instructions.
